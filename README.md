# 🏥 Medical Maintenance Dashboard

Dashboard interactivo desarrollado en Python para el análisis de mantenimiento de equipos médicos. Permite identificar equipos críticos, analizar tiempos de inactividad (downtime), costos de reparación y tipos de fallas para mejorar la toma de decisiones.

---

## 🚀 Objetivo

El objetivo de este proyecto es transformar datos de mantenimiento en información útil para:

- Detectar equipos con mayor impacto operativo
- Analizar costos asociados a fallas
- Identificar patrones de fallas
- Apoyar decisiones de mantenimiento o reemplazo

---

## 🧠 Funcionalidades

- Filtro por tipo de equipo (multiselección)
- Visualización de métricas clave:
  - Downtime total
  - Costo total
  - Promedio de downtime por falla
- Gráficas dinámicas:
  - Downtime total por equipo
  - Tipo de fallas por equipo
- Comparación costo vs downtime
- Generación de insights automáticos

---

## 📊 Ejemplo de uso

El dashboard permite identificar rápidamente qué equipos representan mayor problema.

**Insight clave del dataset:**
> El equipo de Rayos X concentra el mayor downtime y costo acumulado, lo que sugiere priorizar mantenimiento o evaluar estrategias como contratos con proveedor o reemplazo.
